Methodology

The project followed a modular software engineering approach. Individual components were developed and validated independently, then integrated into a single mobile platform.

Each module was driven by its own dataset: speech datasets were used to train NLP pronunciation models; image datasets powered facial emotion recognition; raw sensor readings were collected through IoT devices for stress analysis; and stylus interaction data was used to build motor analytics models.

All machine learning models went through a full pipeline — training, cross-validation, hyperparameter optimisation, and TensorFlow Lite conversion — before being deployed to the mobile application. The modular structure ensured each component could be improved independently without disrupting the overall system.
